Lista de Atajos de VS Code

🔄Navegación entre Paneles

  • Ctrl+1 Enfoca en el editor de código
  • Ctrl+Shift+E Enfoca en Explorer (explorador de archivos)
  • Ctrl+Shift+F Enfoca en Search (búsqueda)
  • Ctrl+Shift+G Enfoca en Source Control (Git)
  • Ctrl+Shift+D Enfoca en Debug
  • Ctrl+Shift+X Enfoca en Extensions
  • F6 Cicla entre todos los paneles
  • Shift+F6 Cicla en sentido contrario

💬GitHub Copilot Chat

  • Ctrl+Shift+I Abre/cierra el panel de chat

🖥️Terminal

  • Ctrl+` Abre/cierra la terminal
  • Ctrl+Shift+` Crea una nueva terminal
  • Ctrl+J Abre/cierra el panel inferior

📄Edición de Código

  • Shift+Alt+Abajo Duplica la línea hacia abajo
  • Shift+Alt+Arriba Duplica la línea hacia arriba
  • Alt+Arriba/Abajo Mueve la línea arriba/abajo

🎯Selección Múltiple

  • Ctrl+D Selecciona apariciones incrementalmente
  • Ctrl+Shift+L Selecciona TODAS las apariciones
  • Ctrl+F2 También selecciona todas las apariciones
  • Ctrl+K Ctrl+D Salta aparición actual y selecciona siguiente
  • Ctrl+U Deshace la última selección múltiple

▶️Ejecutar Código Java

  • Ctrl+F5 Ejecuta sin debugging
  • F5 Ejecuta con debugging
  • Shift+F5 Detener ejecución

🔧Otros Útiles

  • Ctrl+Tab Alternar entre pestañas abiertas
  • Ctrl+Shift+P Paleta de comandos
  • Alt+Tab Alternar entre ventanas
💡 Tip: Los atajos más utilizados son Ctrl+1 (código), Ctrl+Shift+E (explorer), Ctrl+` (terminal) y Ctrl+D (selección múltiple).
